# 📝 Draft Manager

A modern and responsive **Draft Management System** built with **React.js**. This application allows users to create, edit, view, and delete post drafts using React state management.

---

## 📌 Features

- ✍ Create new drafts
- 📋 View saved drafts
- ✏ Edit existing drafts
- 🗑 Delete drafts
- ⚡ Responsive and modern UI
- 🎨 Professional monochromatic theme
- 💻 Built with React Functional Components

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/draft-manager.git
```

Go inside the project

```bash
cd draft-manager
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

Open your browser

```
http://localhost:3000
```
